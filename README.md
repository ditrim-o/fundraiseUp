# fundraiseUp

## Installation

-   install [Node.js](https://nodejs.org/en/download)
-   clone a repository

```sh
git clone https://github.com/ditrim-o/fundraiseUp.git
```

-   go to folder

```sh
cd fundraiseUp
```

-   install npm dependencies

```sh
npm install
```

-   run on localhost

```sh
npm run start
```

-   build application

```sh
npm run build
```
